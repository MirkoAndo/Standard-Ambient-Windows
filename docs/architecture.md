# Standard Ambient Windows - Struttura

Questo progetto organizza lo sviluppo di un ambiente Windows personale con installazione rapida e ripetibile.

## Sezioni

- scripts: script PowerShell per bootstrap e installazione
- config: configurazioni e liste pacchetti (incluse impostazioni)
- tools: strumenti locali o binari scaricati
- logs: log delle esecuzioni
- data: dati di supporto (cache, asset, export)
- tests: test e validazione degli script
- src: moduli riutilizzabili (funzioni, librerie)
- assets: wallpaper e risorse statiche
