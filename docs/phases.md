# Fasi del progetto

## Fase 1 - UI e comfort (visibile)

Obiettivo: rendere l'ambiente gradevole e funzionale dal primo avvio.

- Wallpaper (desktop e lock screen)
- Tema e colori accento con scelta Light/Dark
- Taskbar centrale
- Explorer e visualizzazione: estensioni file e file nascosti visibili
- Notifiche: disattiva tutto tranne Windows Defender
- Windhawk + mod di comfort (serve definire le impostazioni per ogni mod)

### Mod Windhawk (lista)

- explorer-details-better-file-sizes
- taskbar-clock-customization
- taskbar-tray-system-icon-tweaks
- windows-11-notification-center-styler
- windows-11-start-menu-styler
- windows-11-taskbar-styler

Impostazioni Windhawk: [config/settings/windhawk.json](config/settings/windhawk.json)

## Fase 2 - App essenziali

Obiettivo: installare il set minimo di applicazioni per lavorare.

- Browser
- Editor/IDE
- Utility sistema
- Tool base (zip, git, terminal)

Profili di installazione (config/packages.json):

- base: Brave, Discord, WhatsApp, WinRAR, 7-Zip, AnyDesk, Windhawk, Terminal, PowerShell, Git
- dev: VS Code, Python, Java
- gaming: Steam, Driver Booster
- media: vuoto (da definire)

## Fase 3 - Impostazioni di sistema

Obiettivo: privacy, sicurezza e prestazioni coerenti.

- Privacy e telemetry
- Power plan
- Notifiche
- Periferiche e input
- Update policy

## Fase 4 - Produttivita e workflow

Obiettivo: accelerare il lavoro quotidiano.

- Scorciatoie
- Layout e workspace
- Profili terminale
- Script di automazione

## Fase 5 - Backup ed export

Obiettivo: ripristino rapido e portabilita.

- Export configurazioni
- Snapshot pacchetti
- Report finale
